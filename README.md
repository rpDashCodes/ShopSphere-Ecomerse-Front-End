# ShopSphere Frontend

A modern and responsive e-commerce frontend built with React and Vite. ShopSphere provides a smooth shopping experience with product browsing, detailed product pages, search functionality, infinite scrolling, and responsive UI design.

## Features
- Responsive modern UI
- Product listing page
- Product detail page
- Search functionality
- Infinite scrolling / pagination
- Loading screens
- Mobile-friendly navigation menu
- Dynamic routing using React Router
- Reusable component-based architecture
- API integration for fetching products
- Clean folder structure

## Tech Stack
- React
- JavaScript
- React Router DOM
- CSS
- Vite
